Intro

Kita tidak akan bisa bertanggung jawab tanpa mengenali.

Perkenalan

Jaisy merupakan Bahasa Scripting yang dikembangkan oleh Asbab Project untuk menjadi bahasa komunikasi peneliti dalam mengerjakan NLP Rule Based Asbab Project.

Filosofi

Jaisy dibuat untuk mudah dimengerti oleh Pihak yang tidak terbiasa dengan Bahasa Pemprograman. Jaisy menekankan kejelasan algoritma dalam tahapan mengerjakan NLP Rule Based Asbab Project. Oleh karena itu, Jaisy mengorbankan efisiensi penulisan Syntax atau Instruksi. Hal itu membuat Jaisy tampak berbeda dengan Bahasa Pemprograman lainnya. dalam menentukan Aturan Penulisan, Jaisy tidak mempunyai aturan ketat dalam penggunaan bahasa. Jaisy menggunakan bahasa campuran yang dirasa akan mudah dimengerti oleh peneliti.

Syntax Penulisan

Jaisy sangat ketat dalam penulisan syntax. Banyak aturan yang bersifat memaksakan karena menyesuaikan filosofi. Kami menganjurkan pengguna untuk mempelajari dokumentasi dengan seksama.

Editor

Jaisy Editor berbasis Web Based dengan fitur Line Numbering, Syntax Highlight, dan Suggestion. Namun saat ini fitur Syntax Highlight masih menggunakan aturan regex sederhana dan belum dapat mengindikasikan error atau kesalahan syntax. Saat ini masih ada beberapa kekurangan dalam User Experience pada Editor yang membuat kurang nyaman. Harap pemakluman dan dukungannya.

Interpreter

Jaisy merupakan Bahasa Scripting yang membutuhkan interpreter. dalam hal ini, sistem Jaisy Interpreter berjalan di atas bahasa PHP.

Rilis

Kami memutuskan untuk merilis ke Publik untuk membantu mengembangkan NLP Rule Based Asbab Project. Kami belum bisa merilis Jaisy Interpreter ke Publik selama masih tahap pengembangan.

Basic

Syntax dasar dari Jaisy yang harus dimengerti.

Print

Instruksi untuk menampilkan data.
Instruksi Print hanya dapat menampilkan data. Oleh karena itu, Instruksi Print tidak dapat melakukan pemanggilan fungsi dan manipulasi data. 


print Assalamualaina
print جيش 
print
print  Halloo

Assalamualaina جيش Halloo

Data akan ditampilkan secara berurutan dari atas ke bawah. Baris ketiga akan menampilkan baris baru dengan text kosong.

Komentar

Instruksi untuk catatan di dalam kode. Komentar tidak akan dieksekusi sebagai instruksi. Namun hanya digunakan untuk catatan saja. Sehingga instruksi yang diberi komentar tidak akan dieksekusi. Cara menulis komentar adalah dengan menambahkan garis miring kanan // 2 kali. Dapat ditulis di awal baris atau di akhir. 


print Assalamualaina
// kode ini tidak akan dieksekusi
print جيش // jaisy

Assalamualaina جيش

Karena Komentar ada pada awal baris, Instruksi pada baris kedua akan diabaikan secara keseluruhan. Pada baris ketiga, instruksi sebelum komentar akan tetap dijalankan. *Jika kode sudah sangat panjang, komentar sangat berguna untuk menjadi catatan atau menonaktifkan sebagian instruksi.

Transliterasi

Instruksi untuk memanggil karakter Asb Char. Baca List. Cara menulis Indo Char adalah dengan menggunakan @ dengan diikuti kode. Setiap Indo Char hanya dapat memanggil 1 karakter saja. 


print @JIM @YAK @SYIN
print @ALIF @LAM @JIM @YAK @SYIN

جيش الجيش

Data yang ditampilkan akan digabung tanpa spasi, menjadikan data hanya 1 kata.

Pegon

Instruksi untuk merubah text abc menjadi pegon. Cara menggunakan : gunakan karakter pagar. 


print #aku @ #lagi @ #koding
print #Alklam_hw_ALlfZH_Almrkb_bALwDHE

أكو لاڮي كوديڠ الكلام هو اللفظ المركب بالوضع

Gaya penulisan dapat digunakan perkata dengan menggunakan operator penggabungan untuk membuat spasi. Bisa juga dengan menggunakan garis bawah.

Penggabungan Text

Jaisy secara bawaan akan menggabungkan setiap text, dalam artian tidak ada spasi. Jika ingin menambahkan spasi, dapat menggunakan operator penggabungan yaitu, karakter tunggal @/#/$. 


print Assalamualaina
print Hallo Kehidupan
print Hallo @ Kehidupan. # Aku $ Hidup.

Assalamualaina HalloKehidupan Hallo Kehidupan. Aku Hidup.

Pada Baris Ke-3. Data akan ditampilkan tanpa spasi. Pada Baris Ke-4 Data akan ditampilkan dengan spasi. Karena diberi Instruksi Penggabungan.

Text Panjang

Text Panjang adalah data text yang diapit oleh petik tunggal atau petik ganda. Text Panjang akan ditampilkan dengan apa adanya. dalam artian dengan spasi dan segala karakter yang ada di dalamnya. 


print 'ini adalah Jaisy Query Interpreter' @ "Ayo Kapan Ono!!!"
print 'Penulisan Arab : ' @JIM @YAK @SYIN @ JAISYUN

print 'Saya menulis : @ALIF @LAM'

ini adalah Jaisy Query Interpreter Ayo Kapan Ono!!! Penulisan Arab : جيش JAISYUN Saya menulis : @ALIF @LAM

Pada Baris Ke-1 terdapat 2 Text Panjang yang digabungkan. Pada Baris Ke-2 terdapat Text Panjang yang disambungkan dengan Text Pendek yang terdapat Asb Char Pada Baris Ke-4 terdapat Instruksi Asb Char di dalam Text Panjang yang tidak akan terbaca sebagai Asb Char.

Variabel

Variabel adalah instruksi untuk menyimpan data. Cara pendefinisian variabel adalah dengan aturan berikut : ($)nama_variabel (=) Isi_data Variabel tidak dapat didefinisikan 2 kali. Gunakan keyword Update untuk memperbaharui data. 


$nama = faza
$namaArab = @FAK @ALIF @ZAI
print 'Nama saya adalah : ' $nama '. dengan text arab : ' $namaArab

update $nama = ilmi
update $namaArab = @AIN @LAM @MIM
print 'Nama saya adalah : ' $nama '. dengan text arab : ' $namaArab

Nama saya adalah : faza. dengan text arab : فاز Nama saya adalah : ilmi. dengan text arab : علم

Kode tersebut adalah contoh penggunaan variabel dan update. Jika memperbaharui variabel tanpa instruksi update maka akan muncul error. Interpreter akan menganggap adanya pendefinisian ulang variabel.

Skill

instruksi untuk memanipulasi data dan hanya bisa digunakan saat pendefinisian variabel. Cara menggunakan skill : Definisikan Variabel, kemudian masukkan skillnya. Contoh : gunduler : argument1 nama_skill (:) argumen1 (:) argumen2 (:) argumen3 (:) argumen4 


$jaisy       = 'جَيْشٌ'
$jaisyGundul = gunduler : $jaisy
print $jaisyGundul

print 'Jaisy berharokat : ' $jaisy
print 'Jaisy tidak berharokat : ' $jaisyGundul

جيش Jaisy berharokat : جَيْشٌ Jaisy tidak berharokat : جيش

Pada baris ke-2 terdapat skill gunduler yang memiliki argumen yang berisi variabel. *skill gunduler adalah skill untuk menghapus semua harokat.

Fungsi adalah instruksi untuk membuat skill sendiri. Cara menggunakan fungsi yang sudah dibuat seperti menggunakan skill. Namun diberi *. Cara Membuat Fungsi : Buat_fungsi nama_fungsi : $argumen …. tutup_fungsi 


buat_fungsi salam : $var1
$hasil = 'اَلسَّلَامُ عَلَيْكُمْ وَرَحْمَةُ اللهِ وَبَرَكَا تُهُ' @ $var1
tutup_fungsi

Variabel

Variabel mempunyai aturan yang perlu diketahui.

Aturan Variabel

  1. Variabel harus didefinisikan di awal baris.
  2. Nama Variabel tidak boleh mengandung Spasi.
  3. Variabel hanya boleh didefinisikan 1 Kali.
  4. Isi dari Variabel dapat diperbaharui dengan Instruksi update.
  5. Variabel menjadi syarat untuk menggunakan Instruksi Skill dan memanggil Fungsi.
  6. Variabel yang didefinisikan di dalam buat_fungsi tidak dapat diakses di luar buat_fungsi. 


$nama = faza
$namaArab = @FAK @ALIF @ZAI
print 'Nama saya adalah : ' $nama '. dengan text arab : ' $namaArab

update $nama = ilmi
update $namaArab = @AIN @LAM @MIM
print 'Nama saya adalah : ' $nama '. dengan text arab : ' $namaArab

Nama saya adalah : faza. dengan text arab : فاز Nama saya adalah : ilmi. dengan text arab : علم

Pastikan juga membaca Aturan Skill, Fungsi, dan Logika. Karena akan ada banyak contoh penggunaan variabel yang lebih lanjut.

  3. Argument maksimal 4.
  4. Gunakan tanda : untuk pembatas Argument.

Gunduler

Instruksi gunduler adalah instruksi untuk menghapus semua harokat. 


$jaisy = 'جَيْشٌ'
$tesGunduler = gunduler : $jaisy // جيش  
print $tesGunduler

جيش

Variabel $tesGunduler diberi Instruksi SKill gunduler untuk menghapus semua harokat pada variabel $jaisy.

Clean Spasi

Instruksi clean_spasi adalah instruksi untuk menghapus atau membersihkan spasi yang terdapat pada awal dan akhir text. 


$tesCleanSpasi = clean_spasi : ' text  ' // text
print $tesCleanSpasi

text

Variabel $tesCleanSpasi diberi Instruksi SKill clean_spasi untuk menghapus semua spasi di awal dan akhir data text.

Hitung

Instruksi Hitung adalah instruksi untuk menghitung Karakter. 


$jaisy = 'جَيْشٌ'
$tesHitung = hitung : $jaisy         // 12 (harokat dihitung karakter)
print $tesHitung

12

Variabel $tesHitung diberi Instruksi SKill hitung untuk menghitung variabel $jaisy.

Hapus

Instruksi hapus adalah instruksi untuk menghapus semua karakter yang dipilih. 


$jaisyGundul = جيش
$tesHapus  = hapus : $jaisyGundul : @SYIN        // جي
print $tesHapus

جي

Variabel $tesHapus diberi Instruksi SKill hapus untuk menghapus semua karakter @SYIN.

Ganti

Instruksi Ganti adalah instruksi untuk mengganti suatu karakter ke karakter lain.
Argument pertama diisi data yang akan dimanipulasi.
Argument kedua diisi karakter yang akan diganti.
Argument ketiga diisi karakter pengganti. 


$jaisyGundul = جيش
$tesGanti  = ganti : $jaisyGundul : @SYIN : @DAL // جيد
print $tesGanti

جيد

Variabel $tesGanti diberi Instruksi SKill ganti untuk mengganti @SYIN ke @DAL pada variabel $jaisyGundul.

Cari

Instruksi cari adalah instruksi untuk mencari karakter pada text. outputnya berupa angka yang dimulai dari 1. 


$jaisyGundul = جيش

@JIM ada pada huruf pertama dan @SYIN ada pada huruf ketiga.

Tambah

Instruksi tambah adalah instruksi untuk menambah karakter.
Argument pertama diisi data yang akan dimanipulasi.
Argument kedua diisi posisi yang akan ditempati
Argument ketiga diisi karakter yang akan ditambahkan. 


$jaisyGundul = جيش
$tesTambah = tambah : $jaisyGundul : 2 : @WAWU  // جيوش
print $tesTambah

جيوش

Variabel $tesTambah diberi Instruksi SKill tambah untuk menambah @WAWU setelah huruf kedua pada variabel $jaisyGundul.

Ambil

Instruksi ambil adalah instruksi untuk mengambil text yang diseleksi.
Argument pertama diisi data yang akan diambil.
Argument kedua diisi posisi mulai yang akan diambil. (dapat diisi angka atau 'awal' dan 'akhir')
Argument ketiga diisi jumlah karakter yang akan diambil. (dapat diisi angka atau 'semua') 


// contoh ambil dengan argument kedua (posisi mulai) berupa 'awal' dan 'akhir'
$jaisyAbc = jaisy

$tesAmbil1 = ambil : $jaisyAbc : awal : 2 // ja
$tesAmbil2 = ambil : $jaisyAbc : awal : semua // jaisy
$tesAmbil3 = ambil : $jaisyAbc : akhir : 2 // sy
$tesAmbil4 = ambil : $jaisyAbc : akhir : semua // jaisy

print $tesAmbil1
print $tesAmbil2
print $tesAmbil3
print $tesAmbil4

ja jaisy sy jaisy

Variabel $tesAmbil1 mengambil isi variabel $jaisyAbc mulai dari awal, mengambil 2 karakter.
Variabel $tesAmbil2 mengambil isi variabel $jaisyAbc mulai dari awal, mengambil semua sisa karakter.
Variabel $tesAmbil3 mengambil isi variabel $jaisyAbc mulai dari akhir, mengambil 2 karakter.
Variabel $tesAmbil4 mengambil isi variabel $jaisyAbc mulai dari akhir, mengambil semua sisa karakter. 


// contoh ambil dengan argument kedua (posisi mulai) berupa angka. argument ketiga dapat berupa angka positif, angka negatif dan 'semua'.
$jaisyAbc = jaisy

print $tesAmbil7

Variabel $tesAmbil6 mengambil isi variabel $jaisyAbc mulai dari urutan ke-4, mengambil 2 karakter ke belakang.
Variabel $tesAmbil7 mengambil isi variabel $jaisyAbc mulai dari urutan ke-5, mengambil 3 karakter ke belakang.
Variabel $tesAmbil8 mengambil isi variabel $jaisyAbc mulai dari urutan ke-2, mengambil semua sisa karakter.

Antara

Instruksi antara adalah instruksi untuk mengambil text di antara 2 text pembatas.
Argument pertama diisi data yang akan dimanipulasi.
Argument kedua diisi karakter pembatas awal.
Argument ketiga diisi karakter pembatas akhir. 


$jaisyAbc = jaisy
$tesAntara1 = antara : $jaisyAbc : j : i // a
$tesAntara2 = antara : aku dia kamu : aku : kamu // dia

print $tesAntara1
print $tesAntara2

a dia

Pada text 'jaisy', antara j dan i adalah a. Pada text 'aku dia kamu', antara aku dan kamu adalah dia.

Mtk

Instruksi mtk adalah instruksi untuk melakukan Operasi Dasar Matematika. Yaitu Pertambahan, Pengurangan, Perkalian, dan Pembagian. 


$tesMtk1 = mtk : 1 + 2 // 3
$tesMtk2 = mtk : 2 - 1 // 1
$tesMtk3 = mtk : 2 * 2 // 4
$tesMtk4 = mtk : 4 / 2 // 2
print $tesMtk1
print $tesMtk2
print $tesMtk3
print $tesMtk4

// kombinasi
$tesMtk5 = mtk : 5 * 2 + 2 - 3 // 7
print $tesMtk5


// dari variabel
$hitungJaisy = hitung : jaisy      // 5
$tesMtk6 = mtk : $hitungJaisy + 3  // 8

print 'jumlah huruf jaisy ditambah 3 adalah' @ $tesMtk6

3 1 4 2 7 jumlah huruf jaisy ditambah 3 adalah 8

Instruksi mtk juga dapat menerima Argument yang mengandung Variabel.

Logika

Ada beberapa Operator Logika yang dapat digunakan.

Aturan Logika

  1. Mengetahui Instruksi Logika dan Operator Logika
  2. Aksi dari Instruksi Logika hanya bisa digunakan untuk mendefinikan variabel
  3. Dapat digunakan dalam pembuatan fungsi.

Jika Maka

Jika adalah Instruksi untuk menyatakan suatu kondisi (syarat). Maka adalah Instruksi untuk menyatakan suatu aksi (jawab).
Kondisi harus mengandung banding, operator logika dan pembanding. Operator Logika dalam jaisy adalah 'merupakan', 'terdapat', dan 'dan'. 


$input = jaisy
jika $input merupakan jaisy
maka $hasil = 'variabel input adalah jaisy'
print $hasil

variabel input adalah jaisy

Code tersebut melakukan pengecekan. apakah variabel input merupakan jaisy? jika benar, maka definisikan variabel hasil dengan isi tersebut. 


$input = asbab
jika $input merupakan jaisy
maka $hasil = 'variabel input adalah jaisy'
print $hasil

Error baris ke 4 // variabel $hasil tidak ditemukan

Jika pengecekan salah. maka variabel hasil tidak akan didefinisikan. sehingga tidak ditemukan.

Jika Tidak

Instruksi jika_tidak dapat digunakan untuk menyatakan aksi jika suatu kondisi bernilai salah. 


$input = asbab
jika $input merupakan jaisy
maka $hasil = 'variabel input adalah jaisy'
jika_tidak $hasil = 'variabel input bukan jaisy'
print $hasil

variabel input bukan jaisy

Code tersebut terdapat penambahan Instruksi jika_tidak untuk mendefinisikan variabel hasil jika variabel input bukan merupakan jaisy. Sehingga variabel hasil tidak lagi kosong seperti contoh kedua pada section Jika Maka.

Atau Jika

Instruksi atau_jika dapat digunakan untuk menyatakan kondisi alternatif jika Instruksi Jika memiliki kondisi yang bernilai salah. 


$input = asbab
jika $input merupakan jaisy
maka $hasil = 'variabel input adalah jaisy'
atau_jika $input merupakan asbab
maka $hasil = 'variabel input adalah asbab'
jika_tidak $hasil = 'variabel input bukan jaisy atau asbab'
print $hasil

variabel input adalah asbab

Code tersebut terdapat penambahan Instruksi atau_jika untuk mendefinisikan variabel cek sebagai kondisi alternatif, yaitu jika Instruksi Jika memiliki kondisi yang bernilai salah.

Merupakan

Merupakan adalah Operator Logika untuk membandingkan bendanding dan pembanding yang akan bernilai benar, jika banding dan pembanding sama. 


$input = jaisy
jika $input merupakan jaisy
maka $hasil = 'variabel input adalah jaisy'
print $hasil

variabel input adalah jaisy

isi variabel input merupakan jaisy, sehingga bernilai benar.

Terdapat

Merupakan adalah Operator Logika untuk membandingkan bendanding dan pembanding yang akan bernilai benar, jika banding terdapat pada pembanding. 


$input = ai
jika $input terdapat jaisy

Fungsi

Fungsi merupakan pendefinisian skill buatan.

Aturan Fungsi

  1. Fungsi harus didefinisikan sebelum dipanggil.
  2. Fungsi mengsiolasi variabel yang didefinisikan di dalamnya. Sehingga variabel yang didefinisikan di dalam fungsi tidak dapat langsung diakses di luar fungsi.
  3. Fungsi hanya mengeluarkan data yang didefinisikan pada variabel hasil.

Buat Fungsi

Instruksi buat_fungsi harus ditutup dengan Instruksi tutup_fungsi. Instruksi buat_fungsi harus dilengkapi dengan Nama Fungsi dan argument. (maksimal 4 argument). Argument adalah variabel pesanan yang akan diisi ketika pemanggilan untuk kemudian diproses di dalam fungsi. Pemanggilan Fungsi dapat dilakukan dengan penambahan simbol bintang. dan aturannya seperti skill. 


buat_fungsi salam : $var1
$gundulSalam = gunduler : $textSalam
$hasil = $gundulSalam @ $var1
tutup_fungsi

$textSalam = 'ٱلسَّلَامُ عَلَيْكُم'
